# Foundation & Three-Pane Layout Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Set up Tailwind CSS, shadcn-svelte, the dashboard-01 block, then restructure the layout into a sidebar + two resizable panes — using shadcn-svelte components extensively throughout. Every pane should demonstrate real components wired up, not empty divs.

**Architecture:** The sidebar from dashboard-01 becomes the app's left navigation with Collapsible epic tree, Scroll Area, Badge counts, Tooltips, and Dropdown Menus. The main content splits into two Resizable panes: top pane (Gantt area) with Breadcrumb, Toggle Group zoom, Context Menu, Scroll Area; bottom pane (detail panel) with Tabs, Card, Input, Textarea, Date Picker, Slider, Progress, Select, Table, Badge, Kbd, and Empty state. Global overlays use Dialog, Alert Dialog, Command palette, Sheet, and Sonner toast.

**Tech Stack:** SvelteKit 2, Svelte 5 (runes), shadcn-svelte, Tailwind CSS v4, TypeScript, paneforge, lucide-svelte
